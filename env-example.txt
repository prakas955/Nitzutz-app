# - Or rename this file from 'env-example.txt' to '.env'

# Step 2: Get your Gemini API key (FREE!)
# 1. Go to: https://aistudio.google.com/app/apikey
# 2. Sign in with your Google account
# 3. Click "Get API Key" or "Create API Key in new project"
# 4. Copy the generated key (starts with "AIza...")
# 5. Paste it after REACT_APP_GEMINI_API_KEY= above

# Step 3: Save and restart
# - Save this file
# - Stop your dev server (Ctrl+C)
# - Start it again with: npm start
# - Check the browser console for "âœ… Gemini API configured"

# ========================================
# TROUBLESHOOTING
# ========================================

# If you see API errors:
# 1. Make sure your API key is correct (no extra spaces)
# 2. Check if you have internet connection
# 3. Verify your API key hasn't been restricted in Google AI Studio
# 4. Check the browser console (F12) for detailed error messages
# 5. Make sure you have free quota available (60 requests/minute free tier)
